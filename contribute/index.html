Contribute to Protect Black Lives

Want to contribute to this project?

There are so many options! First off, thank you so much for wanting to participate in this project. This collection of resources couldn't exist without the help of people like you. Check out our Submission Guidelines if this is your first time. Protect Black Lives is an open source project. That means that if you're a developer of any skill level, you can contribute. If you aren't a developer, you can still help out!

For Non-Developers

Want to help, but you don't code? No problem! Use this form to submit your resource. Please allow at least 24 hours for your submission to be processed and added.

For Developers

If you don't have a resource but you just want to help, feel free to submit a pull request for any bugs and overall suggestions. Even if it's something as simple as a typo, let us know!

If you have a resource you would like to add, the approach is pretty straight forward! Once you're all set up and ready to submit a pull request to the project repository, follow along with these instructions.

  1. Grab the link to the resource you want to add. Make sure this is a direct link to your content.
  2. After forking the repo, create a new branch for your changes. When naming, use the format [username-resource]
  3. Go to the directory that matches the type of resource you are adding. For example, I'm adding a donation resource, so I will go to donate/index.html.
  4. CTRL + F to find p class="filterLink"
    When you get there, you'll see the code you need to add your resource. Add the link, and change the title accordingly.
    Make sure you're adding your resource to the top of the container, filter-content-container.
  5. Figure out what category and topic(s) your resource belongs to. Here, we filter things by two criteria.
    The type of resource (Donate, Educate, Heal, Petition, or Protest)
    And the topics that resource covers. These are the options that you see on the page that allows you to filter through resources.
    If you have a resource that has multiple types, feel free to add it to both files.
  6. By now, you've probably noticed the span element for the filter buttons. These will allow people to find your resource once you've added it. Add the appropriate tags according to the topics you identified in step 5.
    * If your topic doesn't exist, feel free to create a new one with an appropriate name!
  7. Add a comment above your resource that says Submitted by [your name]

You're done! Submit your pull request and wait for approval.

Submission Guidelines

Generally, most submissions will be accepted as long as they are not hate content, sent with malicious intent and/or plagiarized.

In light of recent events involving peaceful protestors' photos being spread with malicious intent, we take extra precaution to protect them. Resources that include details regarding location of protests and meetups will not be shared because it's been deemed unsafe. If you would like to read more, check out the resources in the Protestor Tech section.

Want to contribute?

Protect Black Lives is an open source project, meaning we would love any and all contributions! If you have a resource that isn't listed that you would like to add, you can read more about how to contribute, even if you don't know how to code!
